# CPU — o que você está segurando

- Chip pequeno, poucos núcleos (4 a 16 em geral)
- Cada núcleo é poderoso: executa tarefas complexas em sequência
- Faz o "raciocínio pesado": rodar o sistema, compilar código, lógica

**Analogia:** um chef de cozinha experiente — faz poucas coisas mas muito bem.

# GPU — o que você está segurando

- Chip maior, centenas ou milhares de núcleos simples
- Faz muitas tarefas simples ao mesmo tempo, em paralelo
- Faz o "trabalho em massa": renderizar pixels, treinar IA

**Analogia:** um batalhão de cozinheiros — cada um faz só uma coisa, mas todos ao mesmo tempo.

<MLToast title="CONEXAO COM IA">
  Treinar uma rede neural = milhões de multiplicações ao mesmo tempo. Só a GPU consegue fazer isso em velocidade útil.
</MLToast>

# Vocabulário técnico — hardware

Termos que aparecem em manuais, benchmarks e fóruns técnicos:

<SlideTable fullWidth>

| Termo EN | Pronuncia | O que significa |
|---|---|---|
| Socket | sóquete | encaixe da CPU na motherboard |
| Slot | slot | entrada para RAM ou GPU (PCI-E) |
| Thermal paste | térmal peist | pasta entre CPU e cooler |
| Benchmark | bênchmark | teste de desempenho |
| Overclock | overclock | rodar peça acima do limite de fábrica |
| VRAM | vi-ram | memória exclusiva da GPU |

</SlideTable>
