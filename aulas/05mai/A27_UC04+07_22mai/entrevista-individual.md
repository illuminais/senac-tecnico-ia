# Entrevista Individual — Shark Tank Tech

> Cada membro responde uma variação diferente. Membro 1 → A, Membro 2 → B, Membro 3 → C, Membro 4 → D.
> O objetivo é o mesmo em todas: testar se a pessoa entende o que o grupo construiu, não só o que o colega falou antes.

---

## Tema 1 — Problema e Cliente

| | Pergunta |
|--|----------|
| **A** | Qual é o problema que a startup de vocês resolve? Descreve em uma frase. |
| **B** | Quem é o cliente de vocês? Por que essa pessoa específica tem esse problema? |
| **C** | Como esse cliente resolve o problema hoje, sem a solução de vocês? |
| **D** | Por que esse problema ainda não tinha uma boa solução antes da startup de vocês? |

---

## Tema 2 — Arquitetura da Solução

| | Pergunta |
|--|----------|
| **A** | O que entra no sistema de IA de vocês e o que sai do outro lado? |
| **B** | O modelo de IA roda na nuvem ou no servidor do cliente? Por que vocês escolheram isso? |
| **C** | Se a internet cair, a solução de vocês para de funcionar? |
| **D** | Descreve o caminho completo em ordem: entrada → IA → saída → quem usa. |

---

## Tema 3 — Modelo de Negócio

| | Pergunta |
|--|----------|
| **A** | Quanto o cliente paga? É por mês, por uso ou por projeto? |
| **B** | Qual é o custo de infraestrutura de vocês por cliente por mês? |
| **C** | A margem é positiva? Quanto sobra depois de pagar a infra? |
| **D** | Qual seria o primeiro cliente real que vocês conseguiriam fechar? Por quê esse? |

---

## Tema 4 — Infraestrutura

| | Pergunta |
|--|----------|
| **A** | A solução de vocês precisa de GPU? Quanto de VRAM e por quê? |
| **B** | Qual modelo de IA vocês usam? É open source ou pago por API? |
| **C** | Quanto custa a infraestrutura de vocês por mês no total? |
| **D** | O dado do cliente fica onde: na nuvem de vocês ou no servidor dele? Qual a diferença? |

---

## Tema 5 — Diferenciação

| | Pergunta |
|--|----------|
| **A** | Por que o cliente não usa só o ChatGPT direto em vez da solução de vocês? |
| **B** | O que a startup de vocês entrega que um desenvolvedor freelancer contratado não entregaria? |
| **C** | Qual é a principal vantagem competitiva da startup de vocês em uma frase? |
| **D** | Se um concorrente lançasse exatamente a mesma coisa amanhã, o que defende vocês? |
